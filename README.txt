# Random Optimization
 
Download repository from https://github.com/Spy7Dragon/Randomized-Optimization

Install for python 3.7
pip install mlrose
pip install numpy
pip install pandas
pip install matplotlib
pip install sklearn

Unzip the download. From root folder of download run 
'py -3 main.py' from a terminal window.

Data is from https://www.lendingclub.com/info/download-data.action

raw_data\LCDataDictionary.xlsx explains the definitions used in the LoanStat3a file. The LoanStat3a file was converted using encodings into lendingclubtraining.csv and lendingclubtest.csv
raw_data\PUMS_Data_Dictionary_2013-2017.pdf explains the definitions used in the original psam_pusa.csv and psam_pusb.csv files used to generate the data.

References:
Hayes, Genevieve. “Algorithms¶.” Algorithms - Mlrose 1.2.0 Documentation, 2019, mlrose.readthedocs.io/en/stable/source/algorithms.html.
